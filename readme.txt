Display Category Posts easily by using basic settings ( No coding knowledge required ).

== Description ==
Call Posts can be used to create posts grid layouts with ease using skin builder ( No coding knowledge required ).

= Features =
* Drag and drop skin builder.
* Responsive design/layouts.
* Supports meta field.
* Clone already saved settings.
* Date range posts filter.
* Filter by post status.
* Post offset(Start with specific post with reference to order number).
* Exclude current posts.
* Adjust Excerpt length.
* Responsive post image(also adjustable).
* Add to widget.
* Media button.

== Installation ==
= Automatic installation =
1. log in to your WordPress dashboard, go to the Plugins menu and click Add New.
2. Search for “Call Posts”.
3. Once found, install by clicking “Install Now” button.
4. Click on activate after installation.
5. A menu will appear after activation( Call Posts ).

= Manual installation =
1. Download the plugin(zip format).
2. Go to Plugin dashboard, Add new and upload the zip folder.
3. Activate the plugin after unzipping/installation.
5. A menu will appear after activation( Call Posts ).
